# Contract Calculator (basic)

Простое приложение для расчета стоимости контрактов в проекте с учетем НДС
Присутствует удобная система анализа, корректирования проектов, и контрактов в них

## Сохранение
По умолчанию все файлы сохранения хранятся в `documents/vat`

## Установка
```bash
pip install -r requirements.txt
```

## Запуск
```bash
python main.py
```

## Формат входного Excel
Таблица должна содержать колонки (в любом порядке, допускаются альтернативные английские имена):
- Название
- Базовая стоимость
- Год начала

## Структура проекта
- core/ - расчёты
- gui/ - интерфейс
- utils/ - работа с Excel и проверки
- data/ - примеры
