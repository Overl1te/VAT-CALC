# Contract Calculator (basic)

Простое приложение для расчёта стоимости контрактов с учётом изменения НДС.
Включает графический интерфейс на Tkinter и сохранение результатов в Excel.

## Установка
```bash
pip install -r requirements.txt
```

## Запуск
```bash
python main.py
```

## Формат входного Excel
Таблица должна содержать колонки (в любом порядке, допускаются альтернативные английские имена):
- Название
- Базовая стоимость
- Год начала

## Структура проекта
- core/ - расчёты
- gui/ - интерфейс
- utils/ - работа с Excel и проверки
- data/ - примеры
